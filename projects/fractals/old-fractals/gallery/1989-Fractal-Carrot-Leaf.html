Fractal Carrot Leaf (1989)

Description

Modelled using an iterated function system (IFS), points inside the disks/spheres are excluded from attractor by a distance calculation, scene ray-traced using rayshade.

Publication

  • Technical Slide Set, SIGGRAPH 1990: 17th Annual Conference on Computer Graphics and Interactive Techniques, Dallas
