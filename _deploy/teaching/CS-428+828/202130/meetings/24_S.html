Consequential Interface Issues
  • Flight 401
  • Therac 25 -- The design did not have any hardware interlocks to prevent the electron-beam from operating in its high-energy mode without the target in place. The engineer had reused software from the Therac-6 and Therac-20, which used hardware interlocks that masked their software defects. Those hardware safeties had no way of reporting that they had been triggered, so preexisting errors were overlooked.
  • (Non-fatal) HODP -- using an interface element in a non-standard way (navigation menu) is problematic for users
  • Slips vs. mistakes
  • Dark UX patterns
