Lab Activity: Scratch

Please use scratch (installed in the UDML, available for download) to create your own implementation of the binary guessing game (described in class). You may use the sample file (startGuessing.sb) as a starting point.

  • Implement a fully-functional version of the game and experiment with some of the multimedia elements.
  • Respond to the usability questions in the text (in Sec. 1.6.1, beginning on page 19)
